PyBuster
========

Aplicacion para la administracion de Video Clubs, desde esta aplicacion se
puede gestionar lo siguiente:

  - Usuarios
  - Clientes
  - Peliculas
  - Prestamos

Colaborar
=========

clonar el repositorio con el comando::
